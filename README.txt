# NudeDetector
### Trabalho 1 de Instrodução ao Processamento de Imagens - CIC/UnB
O trabalho consiste na realização da deteccção de cor de pele em uma dada base de dados de imagens, possuindo uma classe de pessoas com mais pele do que roupa e vice-versa.

### Instruções de execução
 - Os arquivos .m devem ser colocados no mesmo diretório da base de dados;
    - Ex.: /home/user/Dropbox/, considerando a base de dados também no diretório /home/user/Dropbox/.
- O programa que deve ser executado é chamado de _Principal.m_.
